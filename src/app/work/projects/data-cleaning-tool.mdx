Data Cleaning Tool
Overview:
The Data Cleaning Tool is a UI-based Python application designed for efficient data standardization and preprocessing, aimed at helping analysts and data scientists streamline their workflows.

Key Features:

1. Easy-to-use Streamlit interface

2. Automated handling of missing values

3. Duplicate detection and removal

4. Column transformations and type casting

5. CSV and Excel file compatibility

Tech Stack:
Python, Streamlit, NumPy, Pandas

Impact:
Reduced data preprocessing time by automating repetitive tasks, making it especially useful for large datasets in analytics and machine learning projects.

See the project on [GitHub](https://github.com/AbhijeetKumar1505/DCT).
