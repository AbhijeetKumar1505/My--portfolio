Smart Parking Management System
Overview:
A robust web-based system built using Python and Flask for managing urban and corporate parking lots. It provides both user-side booking and admin-side analytics functionalities.

Key Features (User Side):

1. Real-time parking spot availability

2. Instant reservations and booking history

3. Vehicle profile management

4. Parking time estimation

Key Features (Admin Side):

1. Occupancy tracking and real-time dashboards

2. Revenue analytics and lot performance metrics

3. Parking spot and lot management (CRUD)

4. User activity monitoring

Tech Stack:
Python, Flask, SQLAlchemy, PostgreSQL, HTML/CSS, Bootstrap

Impact:
Optimizes space utilization, reduces manual intervention, and enhances the user parking experience while giving administrators full control over operations and analytics.

See the project on [GitHub](https://github.com/24f2007359/parking-system).
