Data Analysis Tool
Overview:
This project is a comprehensive statistical analysis and visualization platform built using Python. It allows users to clean, process, and analyze data with ease.

Key Features:

1. Interactive dashboard for data uploading and preview

2. Summary statistics and correlation matrix generation

3. Graphs: line plots, bar charts, heatmaps, etc.

4. Supports CSV file upload and live previews

Tech Stack:
Python, Streamlit, Pandas, Matplotlib, NumPy

Impact:
Enables non-programmers and analysts to perform EDA (Exploratory Data Analysis) without writing code, bridging the gap between data and decisions.

See the project on [GitHub](https://github.com/AbhijeetKumar1505/Dexter).
